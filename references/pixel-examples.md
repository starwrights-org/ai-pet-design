# 像素风格参考指南

## 经典 8x8 像素 Pet 风格

### Tamagotchi 风格特征
- 极简轮廓
- 大眼睛（占脸部 1/3）
- 圆润形状
- 高对比度（黑白或简单配色）

### Pokémon 早期风格
- 识别度高的轮廓
- 简化但保留特征
- 3-4 色调
- 可爱 + 记忆点

## ASCII 艺术原则

### 基础字符推荐

**轮廓**：
- 圆形：`.-'` `()`
- 方形：`|-` `[]` `┌┐└┘`
- 曲线：`/\` `<>`

**五官**：
- 眼睛：`o O ◉ ● ^^` `. •`
- 嘴巴：`- w v ▽ 3`
- 表情：`^^ >< oo`

**装饰**：
- 阴影：`.,:;`
- 高光：`*+#`
- 纹理：`= ~ ≈`

## 表情系统参考

### 基础表情（推荐先做这 5 个）

**1. 开心**
```
 ^  ^
(  o  )
 \ ▽ /
```

**2. 难过**
```
 •  •
(    )
 \ -- /
```

**3. 惊讶**
```
 O  O
(    )
 \ o /
```

**4. 睡觉**
```
 -  -
(    )
  zZZ
```

**5. 生气**
```
 >  <
(    )
 \ △ /
```

## 尺寸规范

### 推荐尺寸
- **迷你**：4x4 字符（表情图标）
- **标准**：6-8 行（完整 Pet）
- **大型**：10-12 行（详细设计）

---

记住：**简洁 > 复杂**。最好的像素艺术是"一眼就认出来"的设计。
