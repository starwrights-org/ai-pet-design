# Pet ASCII 艺术设计模板

## 设计规范
- 高度：6-8 行
- 宽度：12-20 字符
- 风格：可爱、简洁、易识别
- 字符集：推荐使用 `.-:;+=*#@` 和空格

## 设计1：圆形体型

```
   .---.
  /     \
 |  o o  |
 |   >   |
  \ --- /
   '---'
```

**描述**：圆润、友好型 Pet
**性格关键词**：温和、亲近、治愈系

---

## 设计2：方形体型

```
 ┌─────┐
 │ ◉ ◉ │
 │  ▽  │
 │ ▔▔▔ │
 └─────┘
```

**描述**：方正、稳重型 Pet
**性格关键词**：可靠、踏实、安全感

---

## 设计3：三角形体型

```
    /\
   /  \
  / ^^ \
 /  --  \
/________\
```

**描述**：灵动、活泼型 Pet
**性格关键词**：活力、好奇、探索

---

## 使用说明

1. 复制此模板到新文件：`pet-ascii-designs.md`
2. 修改上述 3 种体型的 ASCII 图案
3. 调整描述和性格关键词
4. 可以添加更多变体（不同表情、动作）
5. 保存并推送到 GitHub
